# Expense Tracker (Python)

A simple Expense Tracker built using Python that allows users to record expenses, store them in a file, and calculate total spending.

## Features

- Add multiple expenses
- Categorize expenses
- Store expense records in a text file
- View stored expenses
- Calculate total expenses
- Exception handling for invalid input

## Concepts Used

- Functions
- Lists
- Dictionaries
- Loops
- File Handling
- Exception Handling
- User Input

## How It Works

1. Enter an expense category.
2. Enter the expense amount.
3. Choose whether to add more expenses.
4. The program stores all expenses in a file.
5. Displays stored expenses.
6. Calculates and displays the total expense.

## Sample Categories

- Food
- Travel
- Shopping
- Entertainment
- Education
